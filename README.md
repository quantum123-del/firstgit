# Listes simplement en C

Ici on ce propose d'implemente quelques fonctions elementaire d'une liste chaine, manipuler eventuelement ces elements.

## Table des Matieres

- [Fonctions propose de la liste](#fonctionnalites)
- [Prototypes de fonctions](#prototypes)
- [Contibuteur](#contributeur)

## Fonctionnalites

- Creer une liste chainee vide
- Verifier que la liste est vide
- Donne la longueur de la liste
- Affiche la liste
- Ajoute un element a la fin de liste
- Ajoute un element au debut de la liste
- Retire le dernier element de la liste
- Retire le premier element de la liste
- Netoyer la liste


## Prototypes

- List push_front_list(List li, int x);
- List pop_back_list(List li);
- List pop_front_list(List li);
- List clear_list(List li);
